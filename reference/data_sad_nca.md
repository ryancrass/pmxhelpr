# Example NCA Parameter Dataset Output from PKNCA

Example NCA Parameter Dataset Output from PKNCA

## Usage

``` r
data_sad_nca
```

## Format

### `data.frame` a data frame with 720 rows and 23 columns:

- ID:

  NONMEM subject identifier

- DOSE:

  Nominal dose assignment (units: mg)

- PART:

  Study part

- start:

  Start of the interval over which parameters are derived (units: hours)

- end:

  End of the interval over which parameters are derived (units: hours)

- PPTESTCD:

  NCA parameter names

- PPORRES:

  NCA parameter values

- exclude:

  Flag for exclusion

- units_dose:

  Dose units

- units_conc:

  Concentration units

- units_time:

  Time units
